Demo Flow

This page describes the review/demo version of getData Remaster. The included local demo app uses sample data only. getData Remaster is operated locally by the administrator and is not distributed as an end-user application.

Actual Data Flow

Participant
  -> voluntarily authorizes Riot Sign On

Riot / Approved API
  -> returns authorized account and custom match data

getData Remaster local admin app
  -> imports/reviews/registers match data locally
  -> creates MVP, rankings, performance, and migration summaries

Telegram bot
  -> displays generated data to authorized users and private group chats

Current Prototype Features

Local Dashboard

Shows private custom game session totals, pending migration candidates, and ranking summaries.

Manual Records

Displays sample custom game records currently entered manually in getData.

MVP & Rankings

Shows sample MVP, ACE, player performance, and point league summaries.

Planned RSO Flow

  1. Participant opens a private Riot account linking URL or flow provided by the getData operator.
  2. Participant is redirected to Riot Sign On.
  3. Participant logs in and explicitly grants access.
  4. getData receives authorized account identity such as PUUID, gameName, and tagLine.
  5. The operator links that Riot account to an internal getData player record.
  6. Authorized match data is used locally to create migration candidates and statistics.
  7. Generated private summaries are shown through the Telegram bot only to authorized bot users and private group chats.

Migration Candidate Concept

Existing getData local record
  player / champion / KDA / level / gold / damage

Authorized Riot match record
  matchId / PUUID / vision score / CS / items / teamId / game duration

Safe local migration result
  legacy + api + resolved fields for administrator review before applying

Included Demo App

The local demo app is located in demo_app/getData_Remaster_RSO_Demo.py. It does not connect to Riot servers. It shows the intended local administrator flow and uses sample data only.

No production API key, RSO client secret, access token, or private player record is included in this package.
